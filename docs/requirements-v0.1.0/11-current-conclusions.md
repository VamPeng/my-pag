# 11. 当前结论摘要

截至 v0.1.0，已确认结论如下：

- 产品仅供个人使用
- V1 以个人使用为主，但后续可能开放产品或源码给他人使用
- 第一版会比极简待办更完整，但不做协作和同步
- V1 在设计取舍上优先保证结构完整，同时允许轻量事项低门槛录入
- 使用树形目录承载工作、生活及项目层级
- 目录在 V1 中主要承担分类作用，不扩展复杂列表语义
- 轻量事项与标准任务共用同一套字段模型，允许部分字段为空
- 状态与进度合并为单一进度字段，采用阶段型表达
- 时间字段仅保留“预期完成时间”
- 优先级保留低、中、高三档
- V1 采用单页面产品结构，桌面端以三栏布局为主
- 快速创建入口放在主界面下方输入框
- 需要提供今天、近期、逾期等时间视图
- 已完成事项不作为单独导航入口，而是通过列表筛选查看
- V1 暂不包含子任务和任务关系能力
- 搜索范围先限制为标题字段
- 共享能力当前不做，但数据层需要为迁移和扩展做准备

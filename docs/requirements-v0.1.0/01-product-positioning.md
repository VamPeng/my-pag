# 01. 产品定位

这是一个面向个人使用的事项管理工具，目标是替代当前用文本文档记录事项的方式，提供更结构化、更易检索、更适合长期管理的记录体验。

该产品不只承载严格意义上的“待办任务”，也允许记录轻量事项，例如：

- 需要和同事确认的事情
- 一时记下的想法
- 还未完全明确的待处理事项

因此，产品在定位上应以“事项管理”作为更上层的概念，其中“待办任务”是最核心的一类事项，但不是唯一类型。

V1 优先解决以下问题：

- 快速记录任何需要记得做的事
- 按工作、生活及项目层级进行整理
- 让不同复杂度的事项都可以被记录，而不是强制填写完整字段
- 通过时间视图快速发现今天、近期、逾期事项
- 通过状态和进展持续跟踪真正需要推进的任务

补充说明：

- 产品核心仍然偏向任务管理，而不是笔记管理
- 对于轻量事项，应允许以更少字段完成记录
- 数据存储方式当前暂不在本章定稿，待技术方案讨论后再确认是否采用本地优先或上云方案

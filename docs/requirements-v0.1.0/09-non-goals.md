# 09. 非目标范围

以下内容明确不纳入 V1：

- 云同步
- 多人协作
- 评论和附件
- 复杂统计报表
- 日历大视图
- 重复任务
- 系统通知提醒
- 基础数据导出
- 子任务
- 任务之间的依赖或关联关系

## 当前明确排除项说明

- V1 不做提醒通知能力
- V1 不做附件、图片或文件上传能力
- V1 不做面向终端用户的登录注册流程
- V1 当前不做上云存储方案
- V1 不做基础数据导出能力

说明：

- V1 需要账号主体，但可以先通过手动方式预置账号
- 当前阶段不要求提供完整登录页、注册页、密码找回等认证流程
- 当前阶段应优先完成单人可用的核心事项管理能力
